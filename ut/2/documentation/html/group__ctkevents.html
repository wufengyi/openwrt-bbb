process_event_t ctk_signal_hyperlink_activate

Emitted when a hyperlink is activated. The signal is broadcast to all listeners.

process_event_t ctk_signal_hyperlink_hover

process_event_t ctk_signal_keypress

Emitted for every key being pressed.

The key is passed as signal data.

process_event_t ctk_signal_menu_activate

Emitted when a menu item is activated. The number of the menu item is passed as signal data.

process_event_t ctk_signal_widget_activate

Emitted when a widget is activated (pressed). A pointer to the widget is passed as signal data.

process_event_t ctk_signal_widget_select

Emitted when a widget is selected. A pointer to the widget is passed as signal data.

process_event_t ctk_signal_window_close

Emitted when a window is closed. A pointer to the window is passed as signal data.
