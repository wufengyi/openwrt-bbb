\subsection{Detailed Description}
The E\+E\+P\+R\+O\+M A\+P\+I defines a common interface for E\+E\+P\+R\+O\+M access on Contiki platforms.

A platform with E\+E\+P\+R\+O\+M support must implement this A\+P\+I.
